# Potrace.js

Potrace vector tracing library Emscripten'd into JS.

## Installing

```
cd potrace
emconfigure ./configure
emmake make

cd ../src
./build.sh
```

## Potrace

Source is available [here](http://potrace.sourceforge.net/), potracelib docs [here](http://potrace.sourceforge.net/potracelib.pdf).
